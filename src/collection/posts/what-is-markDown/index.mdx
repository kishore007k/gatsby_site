# Basic Syntax
| --------------- | -------------------------------- |
|Heading 1        |	# H1                             |
|Heading 2        | ## H2                            |
|Heading 3        | ### H3                           |
| Bold            |	**bold text**                    |
| Italic          |	*italicized text*                |
| Blockquote      |	> blockquote                     |
| Ordered List    |	1. First item                    |
|                 | 2. Second item                   |
|                 | 3. Third item                    |
| Unordered List  |	- First item                     |
|                 | - Second item                    |
|                 | - Third item                     |
| Code            |	`code`                           |
| Horizontal Rule |	---                              |
| Link            |	[title](https://www.example.com) |
| Image           |	![alt text](image.jpg)           |
| --------------- | -------------------------------- |
